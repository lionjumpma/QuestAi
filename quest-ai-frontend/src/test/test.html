在 计算机 编程 语言 中 ， 尤其是 C 语言 ， 编写 一个 “ Hello , World !” 程序 是最 传统的 入门 级 示 例 。 这个 程序 的基本 目的是 向 程序 员 展示 如何 在一个 程序 中 输出 文本 到 控制 台 或 屏幕 。

在 C 语言 中 ， 一个 简单的 “ Hello , World !” 程序 看起来 可能 像 这样 ：

`` ` c

include < std io . h > // 引入 标准 输入 输出 库

int main () { printf (" Hello , World !\ n "); // 打印 " Hello , World !" 到 控制 台 ， 并 换 行 return 0 ; // 表示 程序 顺利 执行 完毕 } `` `

上述 代码 片 断 的工作 原理 如下 ：

1 . # include < std io . h > ： 这一 行 指令 告诉 编译 器 包含 标准 输入 输出 库 （ Stand ard Input / Output library ）， 这是 C 语言 中 用来 进行 打印 （ 输出 ） 和 读 取 （ 输入 ） 等 操作 的 库 。

2 . int main () { ... } ： 在 C 语言 中 ， 每个 程序 都必须 有一个 名为 main 的 函数 ， 程序 的 执行 从这里 开始 。

3 . printf (" Hello , World !\ n "); ： printf 是 C 语言 中 用于 打印 （ 输出 ） 文本 到 控制 台 的一个 函数 。 函数 调用 printf (" Hello , World !\ n "); 会导致 文本 “ Hello , World !” 被 打印 出来 ， \ n 代表 换 行 符 ， 它会 使得 输 出的 文本 之后 会有
